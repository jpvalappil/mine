mine

Semantic-first, zero-dependency UI library.
Include one CSS file and one JS file. Write correct HTML. Get a polished, accessible UI.

Zero dependencies Dark mode ≤ 11 KB gzipped WCAG 2.2 AA

Install

CDN (no build step)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/mine@0.1.0/dist/mine.css">
<script src="https://cdn.jsdelivr.net/npm/mine@0.1.0/dist/mine.js"></script>

npm

npm install mine
import 'mine/dist/mine.css';
import { toast } from 'mine';

Override tokens

:root {
  --mine-color-accent:       #7c3aed;
  --mine-color-accent-hover: #6d28d9;
  --mine-radius-base:        2px;
}

Dark mode

Automatic via prefers-color-scheme. Override with data-theme:

<html data-theme="dark">
<html data-theme="light">
